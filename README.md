# ⚡ Speed Typer

A colorful, motion-rich typing speed test built with **React**, **Vite**, and **Tailwind CSS**.

## ✨ Features

- ⏱ **Time mode** (15s / 30s / 60s / 120s)
- 📝 **Words mode** (10 / 25 / 50 / 100 words)
- 📊 Live WPM, accuracy, and progress tracking
- 🎨 Aurora gradient background with animated blobs
- 🎉 Confetti celebration on test completion
- ⌨️ Visual keyboard that highlights the key you pressed
- 🌈 Gradient flowing text, glass-morphism cards, smooth motion

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## 🛠 Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- TypeScript
